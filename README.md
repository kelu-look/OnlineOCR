# OnlineOCR
This is a web-based application that harnesses the power of PaddleOCR to convert images and PDFs into editable text, offering users an intuitive, responsive interface for accurate text extraction.

## Features

- **PDF & Image Support**: Seamlessly convert both images and PDFs into editable text.
- **Real-time Preview**: Preview documents before initiating the OCR process to ensure accuracy.
- **Dynamic Feedback**: Extracted text is instantly displayed, allowing users to quickly copy or further process the content.
- **Modern UI**: A user-friendly and responsive design ensures a smooth experience across various devices.

## Technologies Used

- **Back-end**: Flask
- **OCR Engine**: PaddleOCR
- **PDF Conversion**: PDF2Image
